# Sublime-Text-3
| 編輯 | 　
| ------------|---------------------------|
| Alt + F3 | 選取所有已選取的單字 |
| Ctrl + / | 註解與移除註解 |
| Ctrl + Shift + / | 區域註解 |
| Ctrl + Y | 回覆上一步，沒有上一步時重複上個動作 |
| Ctrl + Shift + V | 貼上並符合縮排 |


| 通用 | 　
| ------------|---------------------------|
| Ctrl + Shift + P | 命令提示符(包含所有命令) |

| 搜尋/取代 | 　
| ------------|---------------------------|
| Ctrl + F | 搜尋 |
| Ctrl + H | 取代 |
| Ctrl + Shift + F | 檔案間搜尋  |

| 分頁 | 　
| ------------|---------------------------|
| Ctrl + Shift + T | 還原最後關閉的分頁 |
| Ctrl + PgUp | 換到上一個分頁 |
| Ctrl + PgDown | 換到下一個分頁 |
| Ctrl + Tab | 切換上一個分頁 |

| 文字操作 | 　
| ------------|---------------------------|
| Ctrl + K + U | 轉換為大寫 |
| Ctrl + K + L | 轉換為小寫 |
